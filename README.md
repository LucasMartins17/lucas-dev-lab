## 🚀 Projetos em Destaque

### 🔹 1. Pipeline de Dados End-to-End — Indústria de Cinema

**Descrição:** Desenvolvimento de um pipeline completo de Engenharia de Dados na nuvem para tratamento e análise de dados provenientes da API TMDB e arquivos CSV.

**Principais etapas:**

* **Ingestão:** Funções AWS Lambda em Python para extração e armazenamento nas camadas Raw/Bronze no S3
* **Processamento:** Limpeza e normalização com PySpark no AWS Glue
* **Armazenamento:** Conversão dos dados para formato Parquet
* **Modelagem & Analytics:** Consultas SQL no Amazon Athena
* **Visualização:** Dashboards estratégicos no Amazon QuickSight

Projeto com foco em análise de performance de filmes e boas práticas de arquitetura de dados.

---

### 🔹 2. Automação de Processamento de Vendas (Linux)

**Descrição:** Sistema de automação para organização e processamento de arquivos CSV de e-commerce.

**Funcionalidades:**

* Triagem automática de arquivos
* Organização de diretórios (vendas e backup)
* Processamento e validação de dados

**Automação:**

* Agendamento de tarefas para geração de relatórios diários
* Consolidação dos resultados finais

Projeto focado em Shell Script, organização de pipelines locais e controle operacional.

---

### 🔹 3. Conteinerização de Aplicações com Docker

**Descrição:** Implementação de ambientes isolados para desenvolvimento e execução de aplicações.

**Tecnologias utilizadas:**

* Criação de Dockerfiles
* Docker Compose para orquestração de serviços

**Benefícios obtidos:**

* Paridade entre desenvolvimento e produção
* Redução de conflitos de ambiente
* Maior escalabilidade e portabilidade
