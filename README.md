### 🔹 1. Pipeline de Dados End-to-End — Indústria de Cinema

📁 **Acesse o projeto:**
[Ir para o projeto](./Pipeline%20de%20Dados%20Cinema/)

**Descrição:** Desenvolvimento de um pipeline completo de Engenharia de Dados na nuvem para tratamento e análise de dados provenientes da API TMDB e arquivos CSV.

**Principais etapas:**

* **Ingestão:** Funções AWS Lambda em Python para extração e armazenamento nas camadas Raw/Bronze no S3
* **Processamento:** Limpeza e normalização com PySpark no AWS Glue
* **Armazenamento:** Conversão dos dados para formato Parquet
* **Modelagem & Analytics:** Consultas SQL no Amazon Athena
* **Visualização:** Dashboards estratégicos no Amazon QuickSight

Projeto com foco em análise de performance de filmes e boas práticas de arquitetura de dados.

### 🔹 3. Análise de Dados com Python — Google Play Store

📁 **Acesse o projeto:**
[Ir para o projeto](./Analise%20de%20Dados%20Python%20Google%20play/)

**Descrição:** Exploração e processamento do dataset `googleplaystore.csv` utilizando Python para realizar limpeza de dados, cálculos estatísticos e geração de visualizações gráficas para apoio à análise exploratória.

**Tecnologias utilizadas:**

* Python
* Pandas para manipulação e tratamento de dados
* Matplotlib para criação de gráficos e visualizações

**Atividades realizadas:**

* Limpeza e padronização das colunas `Installs`, `Price` e `Reviews`
* Identificação dos Top 5 apps mais instalados
* Análise da distribuição de categorias
* Localização do aplicativo mais caro e do menos instalado
* Contagem de apps classificados como **Mature 17+**
* Top 10 apps por número de reviews
* Cálculo do preço médio por categoria
* Análise da relação entre instalações e reviews

**Benefícios obtidos:**

* Desenvolvimento de habilidades em análise de dados
* Experiência prática com datasets reais
* Melhor compreensão de métricas de mercado de aplicativos
* Consolidação de conceitos de limpeza, tratamento e visualização de dados
